# Task 2: Pinyin Mediation (Phonological Encoding and Hanzi Mapping)

## Overview
This task focuses on the use of pinyin as a phonological mediation system for supporting lexical selection and hanzi production in beginner L2 Chinese learners (CEFR A1–A2).

It builds on manual writing processes and prepares learners for multimodal and AR-supported writing tasks.

---

## Learning Objectives

By the end of this task, learners will be able to:

- Encode Chinese words using pinyin accurately
- Map phonological input to correct hanzi output
- Identify homophone-related ambiguity in Chinese
- Use pinyin as a scaffold for lexical selection and revision

---

## Materials

- Vocabulary list (A1–A2 level)
- Digital device or worksheet
- Pinyin input tool (keyboard or app)
- Previous Task 1 outputs (handwritten characters)

---

## Procedure

### Step 1: Pinyin Encoding
- Convert given vocabulary into pinyin
- Include tone marks where possible

### Step 2: Draft Production
- Write short sentences using pinyin only
- Focus on meaning rather than characters

### Step 3: Pinyin-to-Hanzi Conversion
- Convert drafts into hanzi form
- Compare outputs with reference list

### Step 4: Error Detection
- Identify mismatches between pinyin and hanzi
- Focus on homophones and tone confusion

### Step 5: Peer Exchange
- Compare conversions with a peer
- Discuss differences in lexical selection

### Step 6: Reflection
- What errors were most frequent?
- How does pinyin help or interfere with writing?

---

## Outputs

- Pinyin drafts
- Converted hanzi sentences
- Error log (phonological vs orthographic)
- Peer comparison notes
- Reflection text

---

## Research Layer (Data Generation)

This task generates empirical data on phonological mediation in L2 Chinese writing.

Data types include:

- Pinyin encoding accuracy
- Tone marking consistency
- Pinyin-to-hanzi mapping errors
- Homophone substitution patterns
- Revision and correction cycles

---

## Analytical Use

This task supports:

- Analysis of phonological interference in L2 Chinese
- Comparison with manual writing (Task 1)
- Triangulation with multimodal writing (Tasks 3–4)
- Study of cognitive mediation strategies in lexical selection

---

## Pedagogical–Research Integration

In this task:

- Pinyin production = cognitive scaffolding
- Conversion errors = analytic data
- Revision = learning + research trace

---

## Notes for Teachers / Researchers

This task is essential for observing how learners transition from orthographic encoding (hanzi) to phonological mediation (pinyin), forming the bridge to multimodal writing.
