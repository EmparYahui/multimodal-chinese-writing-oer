# Task 1: Manual Writing (Hanzi Stroke Encoding)

## Overview
This task introduces learners to manual Chinese character writing as a foundational practice for developing orthographic awareness, visuospatial encoding, and stroke order control in beginner L2 Chinese (CEFR A1–A2).

It is part of a hybrid writing sequence integrating pinyin mediation, digital multimodal composing (DMC), and augmented reality (AR)-supported learning.

---

## Learning Objectives

By the end of this task, learners will be able to:

- Produce basic hanzi with correct stroke order
- Recognise radical structures in simple characters
- Develop visuospatial memory for character formation
- Identify and self-correct structural writing errors

---

## Materials

- Printed stroke-order worksheets
- Character set list (A1–A2 vocabulary)
- Writing grid paper
- Pen/pencil

---

## Procedure

### Step 1: Stroke Familiarisation
Observe stroke order and trace characters before writing.

### Step 2: Manual Production
Write each character multiple times focusing on structure and balance.

### Step 3: Error Identification
Identify at least two types of writing errors.

### Step 4: Peer Review
Exchange work and provide feedback.

### Step 5: Reflection
Short reflection on difficulty and strategies used.

---

## Outputs

- Handwriting sheets
- Error annotations
- Peer feedback
- Reflection note

---

## Research Layer

This task generates data on early orthographic development in L2 Chinese.

Data includes:
- Stroke order sequences
- Structural errors
- Revision patterns
- Peer correction traces

These data support qualitative analysis of writing development and can be triangulated with pinyin and multimodal tasks.

---

## Pedagogical–Research Integration

Writing is simultaneously:
- learning activity
- traceable data production
- cognitive process externalisation

---

## Notes

This task forms part of Phase 1 of a four-phase hybrid writing cycle (manual → pinyin → multimodal → reflection).
