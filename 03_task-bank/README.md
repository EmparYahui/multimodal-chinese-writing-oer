# Task Bank Overview

This task bank is part of a hybrid Open Educational Resource (OER) designed to support beginner learners of Chinese (CEFR A1–A2) through integrated handwriting, pinyin mediation, digital multimodal composing (DMC), and augmented reality (AR)-supported situated learning.

The task design is grounded in a project-based learning (PBL) sequence and reflects a multimodal literacy framework in which writing is understood as a distributed, technology-mediated process.

---

## Pedagogical Structure

The tasks are organised into a four-phase learning cycle:

### 1. Manual Writing Phase
- Focus: hanzi stroke order, visual-kinesthetic encoding, orthographic stability
- Output: handwritten character sets + error logs

### 2. Pinyin Mediation Phase
- Focus: phonological encoding, pinyin-to-hanzi mapping, lexical selection
- Output: digital drafts in pinyin + conversion exercises

### 3. Situated Multimodal Phase (AR-supported)
- Focus: contextual language use, fieldwork, spatialised meaning-making
- Output: geolocated artefacts, photos, audio, annotations

### 4. Digital Multimodal Composition Phase
- Focus: DMC integration, rewriting, multimodal coherence
- Output: digital guides, subtitled content, multimodal narratives

### 5. Reflection and Assessment Phase
- Focus: metacognitive awareness, peer/self-assessment, revision strategies
- Output: reflective journals + evaluation rubrics

---

## Design Principles

- Hybrid writing is treated as a continuum, not a separation of modes
- Tasks integrate cognitive, linguistic, and multimodal dimensions
- Revision is central to learning, not an end-stage correction process
- AR and DMC function as meaning-making environments, not add-ons
- Learners are positioned as multimodal authors engaged in situated production

---

## Intended Users

- University-level L2 Chinese beginners (A1–A2)
- Teachers implementing PBL and multimodal literacy approaches
- Researchers in L2 writing, DMC, and technology-mediated learning

---

## Repository Logic

Each task in this bank is designed as a modular learning unit with the following properties:

- Reusable across contexts and proficiency levels
- Adaptable to different institutional settings
- Research traceable through observable production data
- Compatible with multimodal and hybrid literacy frameworks

---

## System Note

This task bank is designed as a component of a broader OER ecosystem combining conceptual framework, learning cycle design, task implementation, and assessment tools. Each element is intended to be independently reusable while maintaining internal coherence within the full system.
