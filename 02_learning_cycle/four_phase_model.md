# Four-Phase Learning Cycle

This learning cycle structures the pedagogical sequence of the OER. It operationalises hybrid writing, multimodal composing, and augmented reality within a coherent developmental trajectory.

---

## Phase 1 — Encoding (Manual Writing)

Focus: orthographic grounding in Chinese characters.

Activities:
- Stroke order practice (hanzi formation)
- Radical decomposition tasks
- Visuospatial memory reinforcement

Output:
- handwritten character sets
- structured stroke accuracy logs

---

## Phase 2 — Mediation (Pinyin → Hanzi Mapping)

Focus: phonological-to-orthographic processing.

Activities:
- pinyin dictation tasks
- homophone discrimination (e.g., 在 / 再)
- guided character selection

Output:
- pinyin drafts
- corrected hanzi conversions
- peer-reviewed revisions

---

## Phase 3 — Expansion (DMC + AR Integration)

Focus: multimodal meaning-making in situated contexts.

Activities:
- geolocated writing tasks
- image + caption production
- audio annotation and subtitle creation

Output:
- multimodal artefacts
- AR-based language entries
- collaborative digital guides

---

## Phase 4 — Reflection (Metacognitive Consolidation)

Focus: revision, evaluation, and awareness of learning processes.

Activities:
- self-assessment
- peer feedback
- revision log analysis

Output:
- reflective journals
- final revised artefacts
- learning progression summaries

---

## Function of the Cycle

The four phases are not linear stages but recursive learning processes in which writing evolves through repeated cycles of mediation, production, and reflection.
